const client = new GuildHub({
  apiKey: 'YOUR_API_KEY',

API'ye Giriş

GuildHub API v3.2 ile geliştirmeye başlayın

Temel Bilgiler

GuildHub API, RESTful tasarım prensiplerine uygun olarak geliştirilmiştir. Tüm API çağrıları HTTPS üzerinden yapılmalıdır ve https://api.guildhub.online/v3 endpoint'i üzerinden erişilebilir.

Kimlik Doğrulama

GuildHub API'yi kullanmak için bir API anahtarına ihtiyacınız var. Anahtarınızı Geliştirici Paneli üzerinden oluşturabilirsiniz. API anahtarlarınızı güvenli tutun ve asla istemci tarafı kodlarınızda paylaşmayın.

Örnek: API anahtarı ile istek 
curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.guildhub.online/v3/users/me

Hata Yönetimi

GuildHub API, standart HTTP durum kodlarını kullanır. Başarılı istekler genellikle 200 durum kodu ile yanıt verirken, hatalar uygun durum kodları ile birlikte bir hata mesajı döndürür.

Kod Açıklama
400 Geçersiz istek (Bad Request)
401 Yetkisiz erişim (Unauthorized)
403 Erişim reddedildi (Forbidden)
404 Kaynak bulunamadı (Not Found)
429 Çok fazla istek (Rate Limit)
500 Sunucu hatası (Internal Server Error)

Hız Sınırlamaları

GuildHub API, kötüye kullanımı önlemek için hız sınırlamaları uygular. Varsayılan olarak dakikada 60 istek gönderebilirsiniz. Hız sınırı aşıldığında 429 durum kodu ile yanıt verilir.
